# Arduino Charlieplexed Dice Roller

Microcontroller coursework project demonstrating Charlieplexing techniques for controlling multiple LEDs using minimal GPIO pins on an Arduino platform.

## Features
- Random dice number generation
- Charlieplexed LED control
- Button-triggered interaction
- GPIO pin minimization using software addressing

## Technologies
- Arduino
- Embedded C/C++
- Charlieplexing logic
